# Bank App - PyQt6 GUI Application

## Overview

This is a banking application built with Python and PyQt6 that simulates basic banking operations including deposits, withdrawals, and balance inquiries. The application has a graphical user interface (GUI) with user authentication and transaction management.

## Features

User Authentication
- Login with username and PIN verification
- Password masking for PIN entry
- User data stored in CSV format
- Clear error messaging for failed login attempts

Banking Operations
- Withdraw: Remove funds from account with balance validation
- Deposit: Add funds to account
- Balance Inquiry: View current account balance
- Transaction confirmation messages

User Interface
- GUI built with PyQt6
- Shows/hides elements based on user actions
- Transaction selection via radio buttons
- Logout functionality to end session
- Input validation and error handling

## Technologies Used

- Python
- PyQt6
- Qt Designer
- CSV

## Learning Objectives

- Object-Oriented Programming: Classes, inheritance, methods
- GUI Development: Event-driven programming, user interface design
- File I/O: Reading from CSV files
- Error Handling: Try-except blocks, input validation
- Data Structures: Lists, dictionaries, indexing
- Software Design: Separation of concerns, modular code
